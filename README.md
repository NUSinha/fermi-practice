# Fermi Problem Practice App

A simple web application for practicing Fermi estimation problems. Test your ability to estimate quantities to the correct order of magnitude!

## Features

- Random Fermi problems from a curated question bank
- Real-time feedback with color-coded results
- Session statistics tracking
- Keyboard shortcuts for quick navigation
- Mobile-friendly design

## How to Use

1. Enter your numerical estimate (supports scientific notation like `5e6` or `3.5E8`)
2. Click Submit or press Enter
3. See how many orders of magnitude off you were
4. Click Next Question to continue practicing
